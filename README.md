  实践 <https://juejin.cn/post/6844904046436843527#heading-17>

## 安装依赖

```
yarn install
```

### 启动vue

```
yarn serve
```

### 安装node依赖

```
  cd serve
  yarn install
```

### 启动node服务

```
  cd serve
  node index.js
```
